# Hospital Appointment Management System

Complete full-stack application with React frontend and Node.js/Express backend.

## Quick Start

1. **Setup Database**: Run `backend/database/schema.sql` and `backend/database/seed.sql` in MySQL
2. **Configure**: Update `backend/.env` with your MySQL credentials
3. **Install Backend**: `cd backend && npm install`
4. **Install Frontend**: `cd frontend && npm install`
5. **Run Backend**: `cd backend && npm start` (port 5000)
6. **Run Frontend**: `cd frontend && npm start` (port 3000)
